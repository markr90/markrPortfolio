About me

Growing up I always loved building Lego, using only a couple blocks you could build infinitely many structures. Software engineering scratches that same itch. Small modular components all coming together to build a large application.

I am a Full Stack Software Engineer with over 2 years of experience working with C#, React and SQL on a complex enterprise web application. My coworkers know me for my good communication and that I am eager to pick up tasks outside my comfort zone. I have experience developing single page web applications and REST APIs using a test driven development approach. With an education in physics I naturally take a data driven approach to solving and analyzing problems.

Outside of work I have several side projects I completed, for example I created a chess game in Java and developed a Chip-8 (CPU) emulator in C#. I also am passionate about cooking, freeride skiing and tennis!
